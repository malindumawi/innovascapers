# innovascapers

🌿 Innovascapers - UI/UX Prototype Design
==================================================

🚀 Project Overview
Innovascapers is a mobile application prototype designed using Figma, focusing on innovative solutions for landscaping, gardening, and smart environmental management. The design emphasizes user-friendly interfaces, accessibility, and modern UI/UX principles.

The project includes a complete interactive prototype along with a demo video showcasing the application flow and features.

--------------------------------------------------
🎯 Objectives
--------------------------------------------------
- Design an intuitive and modern mobile UI/UX
- Provide innovative landscaping and gardening solutions
- Enhance user experience through clean navigation and accessibility
- Demonstrate end-to-end prototype design using Figma

--------------------------------------------------
🛠️ Tools Used
--------------------------------------------------
- Figma (UI/UX Design)
- Proto.io (Prototype export & visualization)

--------------------------------------------------
📱 Key Features (UI Screens)
--------------------------------------------------
✔ Splash Screen with branding  
✔ User Authentication (Sign Up / Login)  
✔ Home Dashboard with feature navigation  
✔ Plants & Trees Information Module  
✔ Products & Equipment Marketplace  
✔ Expense Calculator Dashboard  
✔ Monitor & Control (Smart Garden Monitoring UI)  
✔ Live View (Camera monitoring interface)  
✔ Design Portal (User project visualization)  

(Prototype includes 15+ screens and interactions)

--------------------------------------------------
